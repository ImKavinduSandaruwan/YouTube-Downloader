## INSTALLATION [Termux] :

* `pkg install python -y`
* `pkg install git -y`
* `git clone https://github.com/ImKavinduSandaruwan/YouTube-Downloader-V-1.0`
* `cd cd YouTube-Downloader`
* `python main.py`
